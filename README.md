## About the project
***
A simple web page that was made with the idea of mimicking the design of modern Chinese restaurant/takeout websites to better familiarize myself with HTML and CSS. I found an unappealing looking Chinese takeout website and decided to update it with more modern practices and style like bootstrap. 

## Tech Stack
***
* HTML5
* CSS3

## MVP (Minimum Viable Product)
***
* Two Routes
* Restarant Hours
* Styled with Bootstrap
* Mobile Responsiveness
